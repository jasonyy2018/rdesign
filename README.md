## 🚀 项目简介
**AI职升姬** 是一款基于 AI 大模型的智能化简历制作与职场提升平台。通过先进的 AI 技术，我们致力于帮助求职者突破简历关，实现职场跃迁。

本项目是 **AI职升姬** 的前端部分，基于 Vue3 + TypeScript + Vite + Element-Plus + Pinia 开发。

## 🤼 核心功能
- [x] **AI 智能简历优化**：基于大模型对简历内容进行深度润色与质量诊断。
- [x] **双模设计器**：
  - **在线制作设计器**：专注于高效布局，自动对齐。
  - **积木创作设计器**：极度自由，支持自定义每一个像素。
- [x] **超高清导出**：支持导出可搜索文本的 PDF、JSON 及 Markdown 格式。
- [x] **海量模板库**：提供不同行业、不同风格的高质量简历模板。
- [x] **SEO 专栏**：内置职场百科与求职技巧文章，增强全网搜索可见度。

## 🏊 快速开始
### 1. 克隆项目
```bash
git clone https://github.com/jasonyy2018/rdesign.git
```

### 2. 安装依赖
```bash
npm install --legacy-peer-deps
```

### 3. 本地运行
```bash
npm run dev
```

## 🧱 技术栈
- **核心框架**: Vue 3.4
- **构建工具**: Vite 4.0
- **UI 组件库**: Element-Plus
- **状态管理**: Pinia
- **类型校验**: TypeScript
- **CSS 预处理器**: SCSS
